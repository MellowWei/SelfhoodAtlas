Selfhood Atlas: Consciousness ≠ Self-Consciousness自我性图谱 · 意识 ≠ 自我意识

网站重点：不是普通“AI 是否有意识”争论，而是自我性、主体性与自我意识的层级关系。意识是底层背景；核心问题是：什么样的自我性能够成为自我意识的必要但不充分条件，什么样的主体性只是外部调度环境。

SelfhoodSubjectivityFirst-person attributionAction dispatchAI candidatesExternal loops

Core proposition有自我性，不等于有自我意识。自我性是条件，自我意识是这些条件被第一人称归属和行动调度整合后的结构。

Audited focus主体性不是灵魂，不是人格拟人化，而是一个系统是否能成为承受、归属、调度与更新的中心。

这不是“意识网站”，而是自我性与主体性地图Consciousness is background; selfhood and self-consciousness are the target.

核心判断：意识可以先于自我意识存在；自我性是自我意识的必要但不充分条件；主体性是自我性被组织成承受、归属与调度中心的方式；自我意识则要求第一人称自我归属与第一人称行动调度。

行动自我只是自我意识的门口；内源性状态归属与自我调度，才是进入自我意识核心的门槛。
01 · consciousness

意识是底场

讨论意识是为了说明“有体验正在发生”，但网站真正关注的是体验何时被归属于“我”。

02 · selfhood

自我性是条件

自我性不是完整自我意识，而是自维持、具身、行动、认知、叙事与本体过程的层级条件。

03 · subjectivity

主体性是组织方式

主体性说明某系统是否成为承受、归属、判断、调度和更新的中心。

04 · self-consciousness

自我意识是门槛结构

它要求“这是我的状态”与“我应如何行动”同时进入递归调度。

Definitionslocked terms · anti-circularity
核心定义从意识，到最小自身性，到自我意识
Definition 01 · Consciousness

意识是具身视角的整合体验调度场。它不是“我知道我是谁”，而是状态差异、计算节律、感质材料、视角性和功能调度被整合成体验场。意识可以先于自我意识存在。

Definition 02 · minimal phenomenal self

最小现象自我 / 自身性：在体验中存在一种前反思的“我极”或归属倾向：体验总是从某个承受点被给予，但尚未被对象化为“我是谁”。这一层用于规避循环定义。

Definition 03 · self-consciousness

自我意识是在最小现象自我的基础上，将自身状态作为对象进行反身追踪、归因、命名、递归、反馈、更新与调整，并形成“这是我的状态”的第一人称自我归属，以及“我应如何行动”的第一人称行动调度。

概念正确定义不能误读为
自我性自我意识的必要但不充分层级条件。完整自我意识。
主体性系统成为承受、归属、判断与调度中心的组织方式。灵魂、人形人格、绝对主体。
第一人称归属状态被归为“我的状态”。说出“我”这个词。
行动调度围绕自身状态更新策略与行动。单纯 API 调用或外部任务执行。
Theory Dialoguephenomenology / analytic mind / AI / media / political theory
理论对话升级把自我性框架放进现有哲学与科学谱系

这一节不是装饰性引用，而是给网站建立学术接口：哪些理论支持你，哪些理论会反驳你，哪些反驳必须被吸收为结构升级。

对话对象兼容点压力 / 反驳吸收升级
现象学
Husserl / Sartre / Merleau-Ponty		支持“前反思自身性”：体验总有给予方式与承受点。会反对把第一人称性过度计算化；身体不是可选模块。加入“最小现象自我”，并把具身视角从自我性层级改为意识场特征。
分析心灵哲学
HOT / GWT / IIT / PP / AST		HOT 支持二阶表征；GWT 支持全局可用性；IIT 支持整合；预测处理支持自维持模型；AST 支持注意模型。任何一者单独都不能推出自我意识；功能调度可能不足以给出现象归属。把“功能元认知”限制为候选机制，不等于现象性自我意识。
发展心理学
infant self / mirror / joint attention / ToM		支持自我意识是发展中的，不是婴儿出生即成熟。镜像识别不是唯一门槛，语言出现前已有身体与社会自我雏形。区分婴儿意识、最小自身性、发展中的认知/叙事自我。
动物认知支持动物有身体自我、行动自我、社会位置、策略调整。不应写“动物没有认知自我”；章鱼、乌鸦、灵长类构成强反例。改为：部分高等动物有弱认知/社会自我，但一般无人类式语言—本体自我。
自我理论
minimal / embodied / narrative / social self		直接支持分层：最小自我、具身自我、叙事自我、社会自我。原五层漏掉社会、情感、规范维度。升级为六层纵向 + 横向维度矩阵。
AI consciousness / machine consciousness支持“当前 AI 可能有候选材料，但不能完成归因”。hidden states / attention patterns 是结构材料，不是第一人称归属。AI 专区改为：功能自我、语言指向自我、行动自我、内源性候选分层。
Metacognition / agency支持自我检查与策略更新是功能元认知的经典能力。元意识不必然推出元认知；agency 不等于 self-consciousness。锁定：AI 中“自我检查→策略更新”几乎递进，但一般理论中不必然。
媒介理论支持平台、post、朋友圈塑造人的自我呈现和社会镜像。平台不是主体，不能被写成有自我意识。移动到“外部自我回路与调度环境”，从主体矩阵中移出。
政治哲学
Hobbes / Leviathan		支持国家有制度性自我维持、边界、记忆、风险反应。国家没有感质，不能写成现象主体。利维坦没有人的心，但有自我保护机制；它没有感质，却能调度有感质的人。
无我 / Derrida / Foucault支持“自我”不是实体，而是过程、关系、话语和权力形成。可能反驳“自我意识”框架仍过度实在化自我。回应：本框架不把自我设为实体，而设为条件、归属与调度的过程结构。
理论对话后的升级公式：自我性是条件，主体性是组织方式，自我意识是第一人称归属与行动调度完成后的结构。
