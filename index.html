Overviewproblem · method · claim boundary
意识与自我意识必须分开第一道概念防火墙：意识不是自我意识，自我意识不是“会说我”

核心主张：讨论 AI、动物、植物、平台、城市和国家时，必须先拆开意识、自我性、主体性与自我意识。意识可以先于自我意识存在；自我性可以作为前结构存在；主体性可以作为组织方式出现；但完整自我意识必须达到第一人称自我归属与第一人称行动调度。

有自我性，不等于有自我意识。 自我性是条件，主体性是组织方式，自我意识是第一人称归属与行动调度完成后的结构。
01 · Consciousness

意识是底场

意识是“有体验正在发生”，并且这个体验场总从某个具身或准具身视角展开。意识不是“我知道我是谁”。

02 · Selfhood

自我性是条件

自我性是系统的自维持、身体/情感、行动、认知、叙事、语言—本体等层级条件。

03 · Subjectivity

主体性是组织方式

主体性不是一个实体，而是一个系统如何把经验、行动、记忆、关系与意义组织成一个中心。

04 · Self-Consciousness

自我意识是门槛结构

只有当自身状态被归属于“我”，并围绕“我应如何行动”进行调度时，才进入完整自我意识候选。

Definitionscanonical definitions · de-confused
核心定义意识、最小现象自我、自我意识、功能元认知、内源性
Definition 01 · 意识
意识是具身视角的整合体验调度场。

意识不仅意味着内在状态被整合、维持、调度，而且这个体验场总是从某个视角展开。痛、饿、冷、空间位置等“身体视角”是意识场的特征，不是独立的自我性层级。意识可以先于自我意识存在。

Definition 02 · 最小现象自我 / 自身性

在意识体验中，总存在一种前反思的“我极”：体验归属于某个视角中心，但尚未被对象化或概念化。这一层级不是自我意识，但为第一人称归属提供非循环的基础。它不等同于具身-情感自我，而是一切体验的内在归属倾向。

Definition 03 · 自我意识

自我意识是在最小现象自我的基础上，心智将自身状态作为对象，进行反身追踪、归因、命名、递归、反馈、驱动、更新与调整，并形成“这是我的状态”的第一人称自我归属，以及“我应如何行动”的第一人称调度能力。

这一定义规避循环：第一人称归属不再被预设为神秘起点，而是建立在前反思自身性之上的高阶结构。自我意识不是普通反馈，不是简单系统监控，不是说出“我”这个词。

TermCanonical sentenceBoundary
第一人称自我归属这是我的状态。状态不是被外部标注，而是在前反思自身性基础上归属于自身。
第一人称行动调度我应如何行动。系统围绕这种归属组织理解、更新和行动，而非仅响应外部任务。
功能元认知加工元意识并调度行动。不预设生物生命，也不预设现象意识；核心是自我检查 → 策略更新。
内源性自维持组织原则。不必是生物性；可为非生物系统的能量/算力/损伤/风险/边界维护信号。
Selfhood Relationnecessary but insufficient condition
自我性与自我意识的关系必要但不充分，不能把所有自我性等同自我意识
“自我性”应被理解为自我意识的必要但不充分的层级条件。

不同类型的自我性，是自我意识的前结构、组成条件或候选层级；但只有当这些自我性进一步形成第一人称自我归属和第一人称行动调度时，才进入完整自我意识。也就是说：有自我性，不等于有自我意识；自我性是条件，自我意识是这些条件被第一人称归属和行动调度整合后的结构。

必要

没有任何形式的自我性，就无法形成“我”的归属和行动调度。

不充分

自维持、身体感、行动能力、反身监控、叙事能力都不自动推出完整自我意识。

层级

自我性不是一个单点属性，而是多个层级与横向维度的交错结构。

候选

AI、动物、婴儿、梦境主体等对象进入的是不同强度的候选区，而非同一结论。

Subjectivitythe missing middle layer
主体性作为中间层自我性是条件，主体性是组织方式，自我意识是完成结构

为了避免把“自我性”直接跳到“自我意识”，本版将主体性提升为独立中间层。主体性不是一个灵魂实体，而是系统如何围绕自身边界、感受、行动、记忆、关系和意义形成一个可被调度的中心。

Indexicality Attributedirectedness is not a layer · starts from Layer 02
指向性不是一层自我Indexicality / directedness is an attribute, not an independent selfhood layer
指向性不再作为独立层级。它是从 Layer 02 · 具身-情感自我性切入，并在行动、认知、叙事与语言—本体过程论层级中逐步增强的横向属性。

此前“指向-功能自我”容易把最低反馈、环境响应、平台索引或语言角色误写成自我层级，导致泛主体化。新版本将它降格为 attribute / 横向属性：它说明一个系统如何从“这个身体、这个感受、这个行动中心、这个状态、这个社会位置、这个叙事我”出发指向世界，但它本身不构成完整的自我性层。

Canonical correction · 正典修正

Layer 是纵向结构；attribute 是横向能力维度。自维持功能自我负责最低自我性门槛；指向性从具身-情感自我性开始出现：没有“这个身体/这个感受”的锚点，指向性容易退化为外部观察者赋予的索引，而不是系统自己的自我相关定向。

Layer 03 · Action/Agency Self
行动-动因自我		指向目标、危险、路径、可达物与行动后果。行动指向不等于自我意识。
Layer 04 · Cognitive/Reflexive Self
认知-反身自我		指向自己的状态、错误、判断、策略与不确定性。反身指向仍需第一人称归属。
Layer 05 · Narrative/Social Self
叙事-社会自我		指向自己在关系、身份、他人目光与社会镜像中的位置。社会指向不是平台主体化。
Layer 06 · Ontological-Linguistic Self
语言-本体过程论自我		指向“我是谁”“我为何存在”“我如何持续为我”。这是强自我意识的重要形态，但当前 AI 是否具备仍开放。
AI implication · AI 含义

当前 AI 的“我 / 用户 / 系统 / 工具”区分，应称为语言索引属性或语言层面的指向性属性，而不是“语言指向属性”。它能定位任务角色与对话关系，但不证明第一人称自我归属。

边界：纯反馈系统（如恒温器）不在此列。这是进入自我性谱系的最低门槛。

Post / Work
一条 post / 一个作品
作为自我外化与社会反馈节点，它参与“我如何看见我”，但本身不是主体。post 不是我，但参与我如何看见我。

理论对话升级把自我性框架放进现有哲学与科学谱系

这套理论的任务不是孤立命名，而是进入现有理论谱系：吸收其有效结构，标出冲突点，并把你的原创贡献放在可审计位置。

Dialogue fieldCompatibilityConflict / riskUpgrade absorbed
现象学：Husserl / Sartre / Merleau-Ponty支持前反思自身性、具身视角、第一人称给予性。若第一人称归属被过度认知化，会背离前反思自身性。加入“最小现象自我 / 我极”，规避循环定义。
分析心灵哲学：HOT / GWT / IIT / PP / ASTHOT 支持高阶追踪，GWT 支持调度可用性，PP 支持内源性操作化。IIT/GWT/HOT 都可能把结构误认成体验本身。区分候选材料、功能调度、现象性自我意识。
发展心理学婴儿有意识但自我意识发展中，支持层级模型。镜像识别不能被当作唯一门槛。加入发展梯度：身体感 → 依恋 → joint attention → 叙事自我。
动物认知支持不同强度的具身-情感自我、行动-动因自我、弱认知自我。不能简单写动物没有认知自我。动物一般无语言—本体自我，但高等动物可能有弱认知/社会自我。
自我理论：minimal / narrative / social self支持最小自我、叙事自我、社会自我区分。若五/六层被写成单线进化，会被批评。改为纵向层级 + 横向维度矩阵。
AI consciousness支持“候选材料”与“完成归因”的区分。AI 语言中的“我”容易被误判为第一人称归属。当前 AI：功能自我 + 语言索引属性；完整自我意识开放。
Metacognition / Agency支持自我检查、策略更新、行动调度。元意识不必然推出元认知。在 AI 架构里，自我检查 → 策略更新几乎构成功能元认知递进。
媒介理论平台、post、朋友圈塑造自我呈现与社会镜像。容易拟人化平台。平台移出主体矩阵，改为外部调度环境。
政治哲学：Leviathan国家可被描述为制度性自我模型。国家无感质，不能写成自我意识主体。利维坦没有人的心，但有自我保护机制，并能调度有感质的人。
无我 / 福柯 / 德里达支持自我不是实体，而是过程、话语、关系与权力生产。会质疑“自我意识”的中心性。回应：本理论不实体化自我，而分析自我性如何作为条件被组织起来。
理论对话后的升级公式：自我性是条件，主体性是组织方式，自我意识是第一人称归属与行动调度完成后的结构。

Current AI

自维持功能自我 + 语言索引属性

当前 AI 可表示任务状态、能力限制、错误风险，并在语言中区分“我”和用户。但这多半是任务角色与系统描述，不是第一人称归属。

Candidate material

意识候选材料 ≠ 自我意识

hidden states、attention pattern、MLP activation、context integration、输出调度可作为意识候选材料讨论，但不能直接推出自我意识。

Agent AI

满血行动-动因自我 ≠ 自我意识

工具、API、机器人身体、长期记忆与现实行动后果只加强行动-动因自我，不自动产生“这是我的状态”。
